Working with an XML RSS feed

An RSS (Really Simple Syndication) feed is the technology used when collected data from other websites around the Internet and using them on your own website. For example, on this page the articles are sourced from www.smashingmagazing.com.

This is achived by using the XML HTTPRequest object, which is sent to the server and returns with the requested information. In this case we are using a proxy because we are trying to access the feed from a different domain than the request was made from. This needs to be done because we are using localhost, however in a normal website the request would come from the server hosting the website.

RSS is a very powerfull tool allowing websites to have new content from outside sources without having to update the website manually.
